# MultiAgentSystem Paths

Paths below use `~` to mean the user's home directory. On Windows, `~` resolves to `$HOME` (typically `C:\Users\<username>`).

Canonical framework (this repo):

Wherever you cloned the MultiAgentSystem repository.

Custom agent files (installed):

```text
~/.codex/agents/pm.toml
~/.codex/agents/developer.toml
~/.codex/agents/developer-strong.toml
~/.codex/agents/reviewer.toml
~/.codex/agents/reviewer-strong.toml
~/.codex/agents/researcher.toml
```

Note: `multiagent-orchestrator.toml` was deprecated 2026-06-29 (see `CHANGELOG.md`). It may still be present in old installs; that is harmless.

Useful docs (in the canonical repo):

```text
TEAM-WORKFLOW.md
COMMUNICATION-PROTOCOL.md
ORCHESTRATION.md
CODEX-CUSTOM-AGENTS.md
launch/start-multiagent.md
CHANGELOG.md
FUTURE-PLANS.md
```
